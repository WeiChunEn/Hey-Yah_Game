# HeyYah-Game
ビデオリンク：https://drive.google.com/open?id=1usaKBaGXL5GggqiW6nSb6KJVHg4ZbGJH<br>
這是關於兩隻蝦子在火鍋裡求生存的遊戲<br>
兩位玩家必須一邊躲開上面的障礙物一邊避開對方的干擾<br>
隨著時間的增加
蝦子們的移動速度會越來越快<br>
先逃出鍋子的玩家勝利
若都未逃出則兩敗俱傷<br>
